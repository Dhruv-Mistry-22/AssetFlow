# AssetFlow

**AssetFlow** is an Enterprise Asset & Resource Management System designed to digitize physical asset tracking, allocation, maintenance, and resource booking for any organization.

## 🚀 Core Features

- **Authentication & RBAC**: Secure access for Admin, Asset Manager, Department Head, and Employee roles.
- **Asset Registration**: Complete CRUD operations, auto-generated tags, and full lifecycle tracking.
- **Asset Allocation**: Efficient assignment system with double-allocation prevention and conflict detection.
- **Resource Booking**: Seamless reservation for rooms/vehicles with overlap validation.
- **Maintenance Workflow**: Streamlined process from request creation to approval and resolution.
- **Dashboard KPIs**: Real-time metrics and alerts for better decision-making.
- **CSV Reporting**: Export capabilities for allocations, assets, and maintenance records.
- **Audit Trail**: Comprehensive activity logging and state history for accountability.

## 🛠 Tech Stack

- **Database**: PostgreSQL (via Prisma ORM)
- **Validation**: Zod
- **Framework**: Next.js / Node.js API Routes
